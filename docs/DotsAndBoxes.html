Create a new empty board. A Board is an rectangular grid of dots with spaces that can be filled with lines. Lines can be placed either horizontally or vertically to connect dots. It is implemented as an array of rows Optionally with a specified width and height, otherwise returns a standard 3 dots wide, 3 dots high board.
